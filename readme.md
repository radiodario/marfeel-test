# marfeel-test
> A coding test for marfeel

## grunt-static-site usage:
Initialize the git repository for the project:
```
git init
```

Add and commit the files:
```
git add .
git commit -m 'initial commit'
```

[Create a new repository on github](http://github.com/new)

Make sure it has the same name you entered when you generated this project: marfeel-test

Edit the project files to add your content.

Deploy to GitHub Pages:
```
grunt deploy
```
